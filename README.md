# Street Race X - Frontend UI

Panel de control web e interfaz de usuario para **Street Race X**, la plataforma de emparejamiento equitativo de pilotos y gestión de desafíos callejeros. Desarrollado con una arquitectura modular por componentes, fuertemente tipado en TypeScript y estilizado bajo la identidad visual automovilística utilizando Tailwind CSS.

---

## 🛠️ Tecnologías y Herramientas

* **React 19 & TypeScript** – Núcleo de la interfaz y tipado estricto del dominio.
* **Vite** – Motor de desarrollo y empaquetado ultra rápido.
* **Tailwind CSS v4** – Estilos y diseño responsivo nativo mediante variables CSS.
* **Zustand** – Gestión de estado global ligera para la sesión del piloto (JWT).
* **Axios** – Cliente HTTP base con interceptores automáticos para cabeceras Bearer Auth.
* **React Router Dom v6** – Sistema de enrutamiento dinámico con protección de vistas en el cliente.
